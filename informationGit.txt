cd - переход между папками

git clone (ссылка) - клонирования репозитория на пк


git status - статус проекта (если красное, то проект не отгружен)


git add (файл с форматом) 


git commit -m "изменения" -  добавления файла в репозиторий (2) (комментарий)

git push - выгрузка файла в github 

git pull - загрузка проекта из облака 

git add -A - добавления 2-х файлов в репозиторий


последовательность:

git status
git add
git commit -m
git push
